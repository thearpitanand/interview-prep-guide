# AI/ML 5-Hour Visual Crash Course

## Who This Is For

- Senior SWEs who are beginners in AI/ML
- Visual learners who prefer diagrams over dense text
- Anyone prepping for AI/ML interviews with limited time

## What This Is

A single consolidated guide covering all core AI/ML interview topics — from
fundamentals through deep learning, NLP, and MLOps. Every concept is explained
with diagrams, SWE analogies, and concrete examples. Topics interconnect
naturally rather than living as isolated concept dumps.

## How to Use

Open the complete guide and work through it section by section. Each hour block
is self-contained but builds on the previous ones, so sequential order works
best on a first pass. Expect roughly five hours end-to-end.
